# Paradise Nursery Shopping Application

**Paradise Nursery** is an online plant shop that brings high‑quality houseplants, expert care tips, and a touch of greenery to every home.

## Core Features
- 🌿 Landing page with a nature‑inspired full‑screen background.
- 🪴 Product catalog grouped by categories (Air‑Purifying, Low Light, Low‑Maintenance).
- 🛒 Shopping cart with Redux Toolkit state management.
- ➕ Add to Cart button that instantly updates the cart count.
- 🔢 Quantity controls and dynamic Grand Total calculation.
- 📱 Responsive layout for desktop & mobile.

## Quick Start
```bash
# 1️⃣ Clone the repo
git clone https://github.com/your-username/paradise-nursery.git
cd paradise-nursery

# 2️⃣ Install dependencies
npm install

# 3️⃣ Run the development server
npm run dev
```
Open `http://localhost:5173` in your browser to see the app.
